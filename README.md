# Colecția mea de filme

## Structura proiectului
```
film-react/
├── backend/      ← Express + SQLite
└── frontend/     ← React + Vite
```

## Cum rulezi

### Terminal 1 — Backend
```powershell
cd backend
npm install
node server.js
```

### Terminal 2 — Frontend
```powershell
cd frontend
npm install
npm run dev
```

Deschide: http://localhost:5173
